Potential Based Reward Shaping Based Multidomain RL 포텐셜 기반 보상 쉐이핑 기반 다중도메인 강화학습


  • Tech Stack: Bullet, PyBullet, Blender, Python, Pytorch, Numpy, RL, TL, A2C
  • Github URL: Project Link

Researched and designed a multi-domain reinforcement learning method using potential-based reward shaping. Used Bullet physics simulator and Blender to design diverse environments. Leveraged expert critic networks from multiple simple source domains to shape rewards in more complex target domains. Demonstrated accelerated training and improved generality in diverse environments. 포텐셜 기반 보상 쉐이핑을 이용한 다중 도메인 강화학습 방법을 연구하고 설계했습니다. Bullet 물리 시뮬레이터와 Blender를 사용하여 다양한 환경을 설계했습니다. 여러 단순한 소스 도메인의 전문가 크리틱 네트워크를 활용하여 더 복잡한 타겟 도메인의 보상을 설계했으며, 다양한 환경에서 가속화된 훈련과 향상된 일반성을 입증했습니다.
