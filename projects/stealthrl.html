RL Agent for Measuring the Difficulty of Stealth Game Environment 잠입 게임 환경의 난이도 측정을 위한 강화학습 에이전트


  • Tech Stack: Unity, Python, Pytorch, Stable Baselines3, Numpy, Pandas, Matplotlib, RL, TL, PPO

Designed and implemented stealth game environments with various difficulty using Unity and ML-Agent Toolkit. Developed reinforcement learning agents to quantify and analyze game level difficulty dynamics, leading to enhanced understanding of environment complexity and agent adaptability. Unity 및 ML-Agent Toolkit을 사용하여 다양한 난이도의 잠입 게임 환경을 설계하고 구현했습니다. 강화학습 에이전트를 개발하여 게임 레벨의 난이도 동역학을 정량화하고 분석했으며, 이를 통해 환경의 복잡성과 에이전트의 적응성에 대한 이해를 높였습니다.
