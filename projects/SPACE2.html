Developing robust crowd navigation agents using reinforcement learning with real pedestrian trajectory data. Established environments leveraging Unity and ML-Agents and integrated adversarial mechanisms for generating challenging navigation patterns. Aims to bridge the gap between simulation and real-world navigation in dynamic environments. 실제 보행자 궤적 데이터를 사용하여 강화학습으로 안정적인 Crowd Navigation 에이전트를 개발합니다. Unity와 ML-Agents를 활용한 환경을 구축하고, 도전적인 항법 패턴을 생성하기 위한 적대적 메커니즘을 통합했습니다. 동적 환경에서 시뮬레이션과 실제 세계 사이의 간극을 메우는 것을 목표로 합니다.
